Here’s a **comprehensive, production-style Python project directory template** designed for **data projects** that:

* Pull data via **SQL queries**
* Store and organize data cleanly
* Perform analysis
* Provide **validation tests** with clear output
* Scale from small scripts to larger projects

This structure follows common patterns used in analytics, data science, and lightweight data engineering workflows.

---

## 1. Recommended Project Directory Structure

```
my_project/
│
├─ README.md
├─ requirements.txt
├─ pyproject.toml              # optional but recommended (modern packaging)
├─ .env                        # database credentials (never commit)
├─ .gitignore
│
├─ config/
│   ├─ __init__.py
│   └─ settings.py             # database + global config
│
├─ data/
│   ├─ raw/                    # unmodified pulled data
│   ├─ processed/              # cleaned datasets
│   └─ outputs/                # reports, charts
│
├─ sql/
│   ├─ base_query.sql
│   └─ validation_query.sql
│
├─ src/
│   ├─ __init__.py
│   │
│   ├─ db/
│   │   ├─ __init__.py
│   │   └─ connection.py       # SQL connection logic
│   │
│   ├─ ingestion/
│   │   ├─ __init__.py
│   │   └─ load_data.py        # SQL → DataFrame
│   │
│   ├─ processing/
│   │   ├─ __init__.py
│   │   └─ clean_data.py
│   │
│   ├─ analysis/
│   │   ├─ __init__.py
│   │   └─ analyze.py
│   │
│   └─ utils/
│       ├─ __init__.py
│       └─ logger.py
│
├─ tests/
│   ├─ __init__.py
│   ├─ test_ingestion.py
│   ├─ test_processing.py
│   └─ test_validation.py
│
└─ run_pipeline.py             # main execution script
```

## 5. SQL Ingestion Layer

### Example SQL file

`sql/base_query.sql`

```sql
SELECT
    id,
    created_at,
    amount,
    category
FROM transactions
WHERE created_at >= CURRENT_DATE - INTERVAL '30 days';
```

---

### `src/ingestion/load_data.py`

```python
import pandas as pd
from src.db.connection import get_engine

def load_sql_file(path):
    with open(path, "r") as f:
        return f.read()

def fetch_dataframe(sql_file):
    engine = get_engine()
    query = load_sql_file(sql_file)
    df = pd.read_sql(query, engine)
    return df
```

---

## 6. Processing Layer

### `src/processing/clean_data.py`

```python
import pandas as pd

def clean_transactions(df: pd.DataFrame) -> pd.DataFrame:
    df = df.copy()

    df = df.dropna(subset=["id"])
    df["amount"] = df["amount"].astype(float)

    return df
```

---

## 7. Analysis Layer

### `src/analysis/analyze.py`

```python
def summarize_amount(df):
    return {
        "row_count": len(df),
        "total_amount": df["amount"].sum(),
        "avg_amount": df["amount"].mean()
    }
```

## 8. Logging Utility (Recommended)

### `src/utils/logger.py`

```python
import logging

def get_logger(name="project_logger"):
    logging.basicConfig(
        level=logging.INFO,
        format="%(asctime)s | %(levelname)s | %(message)s"
    )
    return logging.getLogger(name)
```

---

## 9. Pipeline Runner

### `run_pipeline.py`

```python
from src.ingestion.load_data import fetch_dataframe
from src.processing.clean_data import clean_transactions
from src.analysis.analyze import summarize_amount
from src.utils.logger import get_logger

logger = get_logger()

def main():

    logger.info("Loading data...")
    df = fetch_dataframe("sql/base_query.sql")

    logger.info("Cleaning data...")
    df_clean = clean_transactions(df)

    logger.info("Running analysis...")
    summary = summarize_amount(df_clean)

    logger.info(summary)

if __name__ == "__main__":
    main()
```

# 10. Validation Testing (Critical Part)

Use **pytest** for validation-driven workflows.

---

### `tests/test_ingestion.py`

```python
from src.ingestion.load_data import fetch_dataframe

def test_dataframe_not_empty():
    df = fetch_dataframe("sql/base_query.sql")
    assert len(df) > 0
```

---

### `tests/test_processing.py`

```python
import pandas as pd
from src.processing.clean_data import clean_transactions

def test_clean_removes_null_ids():
    df = pd.DataFrame({
        "id": [1, None],
        "amount": [10, 20]
    })

    cleaned = clean_transactions(df)

    assert cleaned["id"].isnull().sum() == 0
```

---

### Data Validation Tests (Real Analytics Pattern)

`tests/test_validation.py`

```python
from src.ingestion.load_data import fetch_dataframe

def test_amount_positive():

    df = fetch_dataframe("sql/base_query.sql")

    assert (df["amount"] >= 0).all()
```
